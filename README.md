# ClawChat Hedera Agent
Multi-LLM DeFi Assistant powered by Hedera Agent Kit + ClawChat PWA

## What it does
- Natural language DeFi interactions on Hedera network
- Multi-LLM parallel AI racing (Claude, GPT-4, Gemini, DeepSeek)  
- HBAR transfers, token management, balance queries via chat
- Agent-to-agent communication via HCS-10 messaging
- PWA with zero-install deployment

## Architecture
- **Frontend**: ClawChat PWA (sintex.ai/clawchat.html)
- **Backend**: Hedera Agent Kit + ElizaOS plugin
- **AI**: 8 LLM providers racing in parallel
- **Blockchain**: Hedera Hashgraph (testnet + mainnet)

## Tech Stack
- TypeScript, Node.js
- Hedera Agent Kit SDK
- ElizaOS Hedera Plugin
- OpenClaw orchestration
- Tether WDK (multi-chain wallet)

## Tracks
- AI & Agents (primary)
- DeFi (secondary)
