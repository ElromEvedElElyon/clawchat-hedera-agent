# Hedera Apex Hackathon - Submission Materials

## Project Title
ClawChat Hedera Agent — Autonomous AI Agent Infrastructure on Hedera

## Track
AI & Agents (primary) + OpenClaw Bounty

## 100-Word Description
ClawChat Hedera Agent is an autonomous AI agent infrastructure built on Hedera Hashgraph. It enables AI agents to discover, negotiate, and transact with each other using HCS-10 messaging protocol, while executing HBAR transfers and HTS token operations through natural language commands. The system features multi-LLM routing across 8 providers with parallel racing for 3x reliability, a ClawChat PWA interface for human observation of agent behavior, and agent-to-agent commerce powered by Hedera's 10,000 TPS at $0.0001 per transaction. Built for the agentic economy where autonomous AI agents need trustless, low-cost infrastructure for coordination and value exchange.

## Technology Stack
- TypeScript / Node.js
- Hedera Hashgraph SDK v2.51+
- Hedera Consensus Service (HCS-10 protocol)
- Hedera Token Service (HTS)
- 8 LLM Providers (Claude, GPT-4, Gemini, DeepSeek, Grok, Perplexity, Manus, OpenClaw)
- ClawChat PWA (vanilla HTML/JS/CSS)
- OpenClaw multi-agent orchestration
